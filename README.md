# 📧 Email Spam Classifier

A machine learning web app that detects whether an email/message is **Spam 🚫** or **Not Spam ✅** using Natural Language Processing techniques.

---

## 🚀 Features

* 🔍 Real-time spam detection
* 🧠 NLP-based text preprocessing (cleaning + lemmatization)
* 🔢 Feature extraction using:

  * Bag of Words (BoW)
  * TF-IDF
* 🤖 Machine Learning model: Naive Bayes
* 🌐 Interactive web app using Streamlit

---

## 🧠 Tech Stack

* Python
* scikit-learn
* NLTK
* Streamlit
* Pandas, NumPy

---

## 🧪 How It Works

1. Input message is cleaned (lowercase, remove symbols, stopwords removal)
2. Lemmatization is applied to normalize words
3. Text is converted into numerical form using TF-IDF
4. Trained Naive Bayes model predicts the result

---

## 📊 Model Performance

* ✔️ High accuracy on spam detection
* ✔️ Handles common spam patterns like:

  * “free money”
  * “win now”
  * “claim prize”
